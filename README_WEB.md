## 🆘 Suporte

### Problemas Comuns

**1. Link não abre**
- Verifique se está logado no Natively
- Tente em modo anônimo
- Limpe o cache do navegador

**2. Login não funciona**
- Verifique email/senha
- Tente recuperar senha
- Confirme que o backend está online

**3. Upload não funciona**
- Verifique tamanho do arquivo (< 10MB)
- Tente outro formato
- Verifique conexão de internet

**4. Dados não aparecem**
- Recarregue a página (F5)
- Verifique se está logado
- Confirme que há dados cadastrados
